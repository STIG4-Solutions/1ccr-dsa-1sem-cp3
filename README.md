## Descrição do sistema

O sistema funciona como uma simulação de caixa eletrônico. O usuário começa com um saque padrão definido como R$ 500.00, podendo realizar saque ou depósito e consultar seu saldo. O código possui validação de input, tornando o sistema blindado contra falhas.

## Funcionalidades implementadas

- Função de consultar saque
- Função de realizar saque
- Função de realizar depósito

## Instruções para compilação e execução

```
gcc main.c
a.exe
```
